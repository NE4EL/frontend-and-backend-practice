О приложении

Версия 1.0.0

Возможности

  • Создание заметок с сохранением в localStorage
  • Напоминания с датой и временем
  • Push-уведомления даже при закрытом приложении
  • Синхронизация между вкладками через WebSocket
  • Офлайн-режим — приложение работает без интернета
  • Устанавливается как PWA на устройство

Технологии

  • Service Worker — кэширование и офлайн-доступ
  • Web App Manifest — установка на рабочий стол
  • App Shell — мгновенная загрузка интерфейса
  • Socket.IO — WebSocket в реальном времени
  • Web Push API — push-уведомления
